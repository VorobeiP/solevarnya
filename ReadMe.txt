========================================================================
    БИБЛИОТЕКА АКТИВНЫХ ШАБЛОНОВ. Обзор проекта DriveEx
========================================================================

Этот проект DriveEx автоматически создан мастером приложений для использования в качестве основы при написании собственной библиотеки динамической компоновки (DLL).

В этом файле представлена сводка содержимого всех файлов, входящих в состав проекта.

DriveEx.vcxproj
    Это основной файл проекта VC++, создаваемый с помощью мастера приложений. Он содержит данные о версии языка Visual C++, использованной для создания файла, а также сведения о платформах, конфигурациях и функциях проекта, выбранных с помощью мастера приложений.

DriveEx.vcxproj.filters
    Это файл фильтров для проектов VC++, созданный с помощью мастера приложений. Он содержит сведения о сопоставлениях между файлами в вашем проекте и фильтрами. Эти сопоставления используются в среде IDE для группировки файлов с одинаковыми расширениями в одном узле (например CPP-файлы сопоставляются с фильтром "Исходные файлы").

DriveEx.idl
    В этом файле содержатся определения библиотеки типов, интерфейсов и компонентных классов, определенных в проекте на языке IDL.
    Этот файл будет обработан компилятором MIDL для создания следующих компонентов.
        Определения интерфейса C++ и объявления идентификаторов GUID (DriveEx.h).
        Определения идентификаторов GUID                                (DriveEx_i.c).
        Библиотека типов                                  (DriveEx.tlb).
        Код маршалинга                                 (DriveEx_p.c и dlldata.c).

DriveEx.h
    В этом файле содержатся определения интерфейса C++ и объявления идентификаторов GUID для элементов, определенных в файле DriveEx.idl. В процессе компиляции файл повторно создается компилятором MIDL.

DriveEx.cpp
    В этом файле содержится карта объектов и реализация компонентов экспорта библиотек DLL.

DriveEx.rc
    Он представляет собой список всех ресурсов Microsoft Windows, используемых программой.

DriveEx.def
    Это файл определений модуля, который предоставляет компоновщику сведения о компонентах экспорта, необходимых для библиотеки DLL. В файле содержатся следующие компоненты экспорта:
        DllGetClassObject
        DllCanUnloadNow
        DllRegisterServer
        DllUnregisterServer
        DllInstall

/////////////////////////////////////////////////////////////////////////////
Другие стандартные файлы:

StdAfx.h, StdAfx.cpp
    Эти файлы используются для построения файла предкомпилированного заголовка (PCH) с именем DriveEx.pch и файла предкомпилированных типов с именем StdAfx.obj.

Resource.h
    Стандартный файл заголовков, в котором определяются идентификаторы ресурсов.

/////////////////////////////////////////////////////////////////////////////
Файл проекта прокси/заглушки DLL и определений модуля:

DriveExps.vcxproj
    Файл проекта, необходимый для построения прокси/заглушки DLL.
	Файл IDL основного проекта должен содержать как минимум один интерфейс. Перед построением библиотеки DLL прокси/заглушки следует скомпилировать файл IDL.
	Этот процесс создает файлы dlldata.c, DriveEx_i.c и DriveEx_p.c, которые необходимы для построения библиотеки DLL прокси/заглушки.

DriveExps.vcxproj.filters
    Это файл фильтров для проекта прокси/заглушки. Он содержит сведения о сопоставлениях между файлами в вашем проекте и фильтрами. Эти сопоставления используются в среде IDE для группировки файлов с одинаковыми расширениями в одном узле (например CPP-файлы сопоставляются с фильтром "Исходные файлы").

DriveExps.def
    Файл определения модуля, в котором содержатся сведения для компоновщика о компонентах экспорта, необходимых для прокси/заглушки.
